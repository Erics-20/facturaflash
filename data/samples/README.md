# Muestras de facturas

Coloca aquí fotos o PDFs de facturas de compra reales para probar la app.

## Requisitos de las imágenes

| Requisito | Detalle |
|-----------|---------|
| Formato | JPG, JPEG, PNG o PDF (una página) |
| Resolución mínima | 1000 × 700 px |
| Iluminación | Uniforme, sin sombras sobre el QR |
| QR | Visible y sin daños — es la fuente de la cabecera |
| Orientación | Derecha (la app no rota automáticamente) |

## Nombres sugeridos

```
F001-00001234.jpg    ← factura F001-00001234
B001-00009999.jpg    ← boleta B001-00009999
```

## Aviso legal

No subas a Git facturas con datos personales o comerciales reales.
Usa imágenes anonimizadas o el Modo Demo de la app para demostraciones.
